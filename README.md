
# FoneMaster - Smartphone Ecommerce

## 🇪🇸 Descripción (Español)
**FoneMaster** es una plataforma de comercio electrónico de alto rendimiento especializada en smartphones de última generación. El proyecto se centra en ofrecer una experiencia de compra premium, con un catálogo interactivo que permite filtrar dispositivos por especificaciones técnicas complejas como memoria RAM y tipo de red (5G/4G).

### Características Principales:
- **Catálogo Dinámico:** Filtrado avanzado y búsqueda en tiempo real.
- **Sistema de Ofertas:** Gestión visual de descuentos y precios originales.
- **Diseño Responsivo:** Optimización total para dispositivos móviles y escritorio.
- **SEO & Social Share:** Metadatos dinámicos para previsualizaciones atractivas en redes sociales.
- **Checkout Flow:** Proceso de compra simulado con validaciones de pasos.

---

## 🇺🇸 Description (English)
**FoneMaster** is a high-performance e-commerce platform specializing in next-generation smartphones. The project focuses on providing a premium shopping experience, featuring an interactive catalog that allows users to filter devices by complex technical specifications like RAM and network type (5G/4G).

### Key Features:
- **Dynamic Catalog:** Advanced filtering and real-time search.
- **Sale System:** Visual management of discounts and original pricing.
- **Responsive Design:** Full optimization for mobile and desktop devices.
- **SEO & Social Share:** Dynamic metadata for attractive social media previews.
- **Checkout Flow:** Simulated multi-step checkout process.

## 🛠️ Tecnologías / Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Frontend:** React 19
- **Styling:** Tailwind CSS & ShadCN UI
- **Components:** Radix UI
- **Icons:** Lucide React
- **Language:** TypeScript
- **AI Tooling:** Genkit
